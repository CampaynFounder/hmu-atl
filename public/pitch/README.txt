HMU CASH RIDE — /pitch video drop-in folder
=============================================

Drop your 9:16 vertical videos here using these exact filenames.
The page will auto-detect them on load. Missing files show a
"DROP VIDEO" placeholder inside the phone frame.

Recommended specs
-----------------
- Aspect ratio:  9:16 (vertical)
- Resolution:    1080 x 1920
- Codec:         H.264 (MP4)
- Audio:         muted / no audio (page autoplays muted)
- Duration:      15–25s per chapter, 45–60s for hero
- File size:     < 10 MB each (Cloudflare Worker friendly)
